# Open Gallery

**Open Gallery**는 작가와 작품을 등록하고, 전시 정보를 관리할 수 있는 Django 기반 웹 애플리케이션입니다. REST API를 제공하며, 이미지 처리 및 PostgreSQL을 데이터베이스로 사용합니다.

## 주요 기능

- 작가 등록 및 관리
- 작품 등록 및 조회 (이미지 포함)
- 전시 등록 (제목, 시작일, 종료일, 작품 목록 입력 가능)
- REST API 지원
